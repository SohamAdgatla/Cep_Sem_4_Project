# Personal Finance Analyzer Notes

## Project Overview
This is a Streamlit-based personal finance health analyzer that uses proven financial formulas to assess financial health.

## Key Features
- Formula-based financial calculations
- Interactive web interface
- Real-time analysis
- Personalized recommendations
- Educational content about financial formulas

## How to Run
```bash
streamlit run main.py
```

## Financial Formulas Used
- Savings Rate: (Income - Expenses) / Income × 100
- Debt-to-Income Ratio: Debt Payments / Income × 100
- Credit Utilization: Balance / Limit × 100
- Emergency Fund: Savings / Monthly Expenses
- Debt-to-Asset Ratio: Debt / (Debt + Savings) × 100
